# Legacy MQL GPU (herança)

O diretório `legacy/` mantém o código original do projeto GPU utilizado antes da
reestruturação atual. Ele está dividido em:

- `legacy_mql_gpu_original_dev/` – árvore histórica com fontes, scripts e
  documentação antigos.
- `legacy_mql_gpu_original_runtime/` – cópia correspondente do runtime da época.

Esses arquivos são apenas para consulta. **Não** os utilize para builds,
deploys ou ajustes do fluxo atual (`Dev/` → `runtime/`). Todo o desenvolvimento
ativo acontece nas pastas `Dev/Experts`, `Dev/Indicators`, `Dev/Include`,
`Dev/Scripts`, `Dev/bin` e scripts associados.
